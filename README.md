## Старт
1. Клонировать репозиторий: 
   ```
   git clone git@github.com/jelt0n/trainscan_test.git
   cd trainscan
   ```
2. Создать виртуальное окружение: `conda create -n trainscan python=3.10 -y`
3. Активировать: `conda activate trainscan`
4. Установить зависимости: `pip install -r requirements.txt`



# Api + UI
1. Запускаем api.py:
```
python api.py
```
3. Переходим по ссылке `http://localhost:8000`

4. Документация по сслыке `http://localhost:8000/docs`
